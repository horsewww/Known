## 详细内容

**1. RadioList**

- 支持平原（Plain）模式

```csharp
protected override void BuildRenderTree(RenderTreeBuilder builder)
{
    builder.Field<RadioList>("平原模式：", "RadioList").Value("3")
            .Set(f => f.IsPlain, true)
            .Set(f => f.Items, Items)
            .Build();
}
```
