# V1.2.0

## 更新日期

- 2023-05-13

## 更新内容

- 😄基于C#和Blazor实现的快速开发框架（前后端分离、跨平台、多端运行）
- 😄分三个类库
    - Known为前后端共用库
    - Known.Core为后端库
    - Known.Razor为前端库
- 😄功能：模块、角色、用户、日志、授权、定时任务
- 😄支持.NET7
